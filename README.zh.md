# JAV Archivist

用于管理 JAV 视频收藏的 Windows 桌面应用。自动扫描文件夹注册视频，并从网络获取元数据以丰富您的媒体库。

**其他语言版本：** [English](README.md) · [한국어](README.ko.md) · [日本語](README.ja.md)

---

## 主要功能

- **媒体库扫描** — 递归扫描指定文件夹，自动注册视频文件
- **元数据抓取** — 根据番号自动获取标题、演员、系列、厂商和标签
- **缩略图 / 样本图片** — 使用 Windows Media Foundation 提取帧，无需外部工具
- **进度条预览** — 视频播放器中的时间轴缩略图预览
- **筛选 & 搜索** — 按演员 / 系列 / 厂商 / 标签 / 收藏 / 观看状态等多种条件筛选
- **演员 · 系列 · 标签管理** — 在专属页面浏览元数据
- **收藏 / 观看记录** — 按视频追踪状态
- **日志** — 实时查看应用运行记录（需在设置中启用）

## 安装

> **尚无发布版本？** 请参阅下方的[从源码构建](#从源码构建)章节。

从 Releases 页面下载 `.msi` 或 `.exe` 安装程序并运行。

---

## 快速开始

1. **设置 → 扫描文件夹** — 添加视频所在的文件夹。
2. **媒体库 → 扫描** — 扫描文件夹以注册视频。
3. **抓取** — 选择视频或运行批量抓取以填充元数据。
4. **浏览** — 按演员 / 系列 / 标签等条件筛选，管理您的收藏。

---

## 界面说明

| 标签页 | 说明 |
|--------|------|
| 媒体库 | 以网格视图浏览视频列表，支持搜索和筛选 |
| 演员 | 浏览注册的演员列表，按演员筛选视频 |
| 系列 | 按系列对视频进行分组 |
| 标签 | 标签列表及关联标签分析 |
| 厂商 | 按制作公司筛选视频 |
| 设置 | 扫描文件夹、抓取选项、日志设置 |
| 日志 | 实时查看应用事件（需在设置中启用） |

## 从源码构建

### 前置条件

- [Node.js](https://nodejs.org/) 18 或更高版本
- [pnpm](https://pnpm.io/) (`npm install -g pnpm`)
- [Rust](https://www.rust-lang.org/tools/install)
- [Tauri 前置条件](https://tauri.app/start/prerequisites/)（包含 Visual Studio C++ 构建工具）

### 构建

```bash
# 安装依赖
pnpm install

# 以开发模式运行
pnpm tauri dev

# 生产构建（输出至 src-tauri/target/release/bundle/）
pnpm tauri build
```

## 技术栈

- **前端** — React 19, TypeScript, Vite, TailwindCSS, Zustand
- **后端** — Rust, Tauri 2, SQLite (rusqlite)
- **媒体处理** — Windows Media Foundation（缩略图与帧提取）
- **抓取** — rquest, scraper
