# JAV Archivist

JAV動画コレクションを管理するためのWindowsデスクトップアプリです。フォルダをスキャンして動画を自動登録し、オンラインからメタデータを取得してライブラリを充実させます。

**他の言語で読む:** [English](README.md) · [한국어](README.ko.md) · [中文](README.zh.md)

---

## 主な機能

- **ライブラリスキャン** — 指定したフォルダを再帰的に探索して動画ファイルを自動登録
- **メタデータスクレイピング** — 品番をもとにタイトル、出演者、シリーズ、メーカー、タグを自動収集
- **サムネイル / サンプル画像** — Windows Media Foundationでフレームを抽出。外部ツール不要
- **スプライトシークバー** — 動画プレイヤー内のシークバーにプレビューを表示
- **フィルター & 検索** — 出演者 / シリーズ / メーカー / タグ / お気に入り / 視聴状況などで絞り込み
- **出演者 · シリーズ · タグ管理** — 専用ページでメタデータを閲覧
- **お気に入り / 視聴履歴** — 動画ごとにステータスを管理
- **ログ** — アプリの動作履歴をリアルタイムで確認（設定から有効化）

## はじめ方

1. **設定 → スキャンフォルダ** — 動画が保存されているフォルダを追加します。
2. **ライブラリ → スキャン** — フォルダをスキャンして動画を登録します。
3. **スクレイピング** — 登録した動画を選択するか、一括スクレイピングを実行してメタデータを入力します。
4. **ブラウズ** — 出演者 / シリーズ / タグなどで絞り込みながらコレクションを管理します。

## 画面構成

| タブ | 説明 |
|------|------|
| ライブラリ | グリッドビューで動画一覧を閲覧、検索・フィルター |
| 出演者 | 登録された出演者一覧、出演者ごとの動画フィルター |
| シリーズ | シリーズごとに動画をグループ化 |
| タグ | タグ一覧と関連タグの分析 |
| メーカー | 制作会社ごとの動画フィルター |
| 設定 | スキャンフォルダ、スクレイピングオプション、ログ設定 |
| ログ | アプリイベントをリアルタイムで確認（設定から有効化が必要） |

## ソースからビルド

### 事前準備

- [Node.js](https://nodejs.org/) 18以上
- [pnpm](https://pnpm.io/) (`npm install -g pnpm`)
- [Rust](https://www.rust-lang.org/tools/install)
- [Tauri前提条件](https://tauri.app/start/prerequisites/)（Visual Studio C++ビルドツールを含む）

### ビルド

```bash
# 依存関係のインストール
pnpm install

# 開発モードで実行
pnpm tauri dev

# 本番ビルド（src-tauri/target/release/bundle/ に生成）
pnpm tauri build
```
